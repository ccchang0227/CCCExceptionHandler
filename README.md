## Installation

CCCExceptionHandler is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CCCExceptionHandler', :path => '~/Desktop/iOS/Pods/CCCExceptionHandler'
```
